# BizDaily — Scaffold (App Router)

This branch contains a scaffold for the BizDaily hackathon MVP:
- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- shadcn/ui (UI primitives)
- Lucide React (icons)
- Recharts (charts)

This scaffold contains routes and placeholder components for:
- Landing Page
- Home Dashboard
- Record Today's Business
- Debt Management
- Business History
- Generate Reports
- Settings

How to use
1. Install dependencies (next, react, tailwind, etc).
2. Wire up Zustand and implement stores in stores/*.ts.
3. Implement business logic in utils/bizLogic.ts.
4. Seed demo data from data/demo-data.ts.
